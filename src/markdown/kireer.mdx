# 背景

6—15 岁是孩子的职业启蒙黄金期。但也有调查显示，孩子在 3 岁左右就开始展现自己的个性和爱好，作为家长我们需要帮助孩子激发他的潜力和优势，进行简单适量的职业认知训练。我们选择了 5-10 岁的儿童，真对他们的认知水平和兴趣爱好进行设计。

# 介绍

Kireer 是一个以职业启蒙和平等教育为价值核心的 AR 教育类平台。儿童会以游戏的方式体验每个职业，通过体感、遥控或者 AR 的形式来进行角色扮演，完成设定的职业任务。

# 职能

这是一个团队项目，我负责了界面的高保真设计、交互的高保真设计以及 Kireer 的 LOGO 设计。在这个项目中，我希望能够尽量模拟真实的交互效果，所以使用了大量的动效与智能动画。

我们还探索了使用 AI 来生成我们的儿童职业图像。我们自己训练了 Lora，让最终的图像符合我们的要求。
