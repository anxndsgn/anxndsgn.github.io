# 背景

AI 的发展给视障人群带来了新的希望。但是现有的 AI 技术主要用于辅助驾驶和人脸识别，对于视障人群的生活帮助并不多。我们希望通过 AI 技术来帮助视障人群。

# 介绍

TouchSee 是一款帮助视障人群感受照片的应用程序。通过分割模型 SAM 与注意力模型，用户可以通过手机不同的震动频率感受照片的每个角落。通过 LLM 与 TTS，用户可以听到对于照片的详细描述。我们希望 AI 技术除了可以用来辅助驾驶和人脸识别，还可以用于帮助视障人群和其他残疾群体，使其获得与普通人感受环境的能力。

# 挑战

TouchSee 的设计与普通的交互设计不同，我们需要根据视障人群的习惯来设计。普通人可以通过眼睛快速获取屏幕上的大部分信息，而通过调研发现，视障人群通过旁白功能来与手机交互，是线性的交互，需要从上到下逐个感受信息。因此我们在信息架构上进行了仔细设计，使得视障人群的用户体验更加流畅。一方面，我们只保留了需要频繁操作的按钮，将不常用的操作集成到了更深的页面中；另一方面我们通过语音提示来引导用户操作。
