# 背景

当前的购物软件存在购物车功能设计不足的问题。当购物车中的商品过多时，用户很难找到他们想要的商品。例如，用户可能需要购买组装台式机、装修材料或者是在优惠期间凑单商品，但需要在长长的购物车列表中进行选择。更糟糕的是，当用户退出或者刷新页面时，购物车中的选择会丢失，需要重新选购。

# 介绍

我重新思考了购物车的设计概念，并引入了现实生活中常见的购物清单的概念。在新的设计中，用户可以在购物车中创建多个购物清单，将商品按照不同的分类进行整理，从而方便用户查找所需商品。用户可以在购物车中查看所有的商品列表，也可以查看单个购物清单中的商品。在商品详情页中，用户可以通过长按购物车按钮将商品添加到购物清单中。此外，用户还可以根据顶部清单标签快速筛选购物车中的商品。
